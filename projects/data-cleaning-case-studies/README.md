# Data Cleaning & Transformation Case Studies

## Overview
Examples of fixing messy data: missing values, mixed types, duplicates, bad dates, and inconsistent text.

## Cases (planned)
1. Customer addresses — standardize states/ZIP, deduplicate.
2. Transactions time — parse timestamps, fix timezones.
3. Product catalog — unify categories, handle nulls.

## Tools
Python (pandas, numpy), SQL (CTEs, window functions), regex.

## Artifacts
- `datasets/`: before/after CSVs
- `scripts/`: Python/SQL examples
